Mobile-first Design

Smartphones, Smartwatches, and Tablets have become the predominant way people interact with the web. Creating Web pages, and Applications that work consistently across every device is fundamental to the success of any Application or Web site. Optimizing your content for the modern web is no longer optional, adopting a Mobile-first approach is essential.

Development

As a Full-stack developer, I'm knowledgeable in every level of how the web works. Setting up and configuring Linux servers, writing server-side API, designing the front-end javascript to power an App, and everything in between. Hype based development is a real thing, developers hear about the flavor of the week and then attempt to shoehorn a project into a mold that it was not meant to fill. I leverage my knowledge of Modern Programming languages, and tools to ensure your project is the best that it can possibly be.

Analytics

Data is king. Knowing how to use the data generated by users using your site or applications is essential to providing smarter, and more successful services. Search Engine Optimizing or SEO, helps you reach a wider audience by making it more search engine friendly and increasing your position on applicable search results. But what good is any of this data if you can't make sense of it? I provide analytic dashboards customized for you, so you can visually interpret the data that means the most to you.

Mother's Day Travel App

I Love my Mother very very much. My mom loves to travel, but after suffering an injury at work, my mom hasn't been able to travel anywhere. Since I couldn't physically take her to all the places she would love to visit, I decided to Virtually allow her to travel to these places. Utilizing the API from unsplash.com I was able to design a simple minimalist frontend using Javascript, PUG, and SASS so that my mom could travel around the world without leaving her own house.
